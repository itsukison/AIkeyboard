# プライバシーポリシー

**最終更新日：2026年6月7日**

株式会社Core7（以下「当社」といいます。）は、当社が提供するスマートフォン向けキーボードアプリ「敬語なう」（以下「本アプリ」といいます。）におけるユーザー情報の取扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」といいます。）を定めます。

本アプリは、ユーザーが入力した文章を、敬語・ビジネス文・メール文・翻訳文・言い換え文などに変換することを目的としたAIキーボードアプリです。当社は、キーボードアプリという性質上、ユーザーの入力内容には個人的・機密的な情報が含まれ得ることを十分に認識し、必要最小限の情報のみを、明確な目的の範囲内で取り扱います。

当社は、ユーザーが安心して本アプリを利用できるよう、透明性の高い説明、適切な安全管理、不要なデータ収集の抑制に努めます。

## 1. 基本方針

当社は、本アプリの提供にあたり、以下の方針を重視します。

1. **ユーザーが明示的にAI機能を利用した文章のみを処理します。**
   本アプリは、ユーザーが「敬語」「メール」「翻訳」「言い換え」等のAI機能をタップして送信した文章を処理します。

2. **ユーザーが入力するすべての内容を常時記録することはありません。**
   本アプリは、ユーザーのすべてのキーボード入力、すべての打鍵、すべての会話内容を常時収集・保存するものではありません。

3. **パスワード、クレジットカード番号、認証コード等の機密情報の収集を目的としていません。**
   iOSの仕様上、パスワード欄等の安全な入力欄では標準キーボードに切り替わる場合があります。また、当社はそのような機密情報の取得を意図していません。

4. **AI変換に必要な範囲で、ユーザーが送信した文章を処理します。**
   AIによる文章変換を行うためには、ユーザーが選択・送信した文章を当社または当社の委託先のAI処理基盤に送信する必要があります。

5. **サービス改善のためのデータ利用は、個人が直接特定されにくい形で行います。**
   当社は、AIの品質改善、機能改善、不具合調査、利用傾向の分析等のため、必要に応じて文章変換に関するデータを利用する場合があります。その際は、可能な限り個人を直接特定できない形で取り扱います。

## 4. フルアクセスについて

iOSのカスタムキーボードでは、インターネット通信を利用する機能を提供するために「フルアクセス」の許可が必要となる場合があります。

本アプリでは、AIによる文章変換、翻訳、メール文作成、言い換え等の機能を提供するため、ユーザーが明示的に送信した文章を当社または当社の委託先のサーバーへ送信する場合があります。そのため、一部のAI機能の利用にはフルアクセスが必要となることがあります。

ただし、フルアクセスを許可した場合であっても、当社はユーザーのすべての入力内容を常時収集するものではありません。当社が処理するのは、原則としてユーザーがAI機能の利用のために明示的に送信した文章、および本アプリの運用に必要な技術情報に限られます。

ユーザーは、iOSの設定画面からいつでも本アプリのフルアクセスを無効にすることができます。フルアクセスを無効にした場合、一部のAI機能が利用できない、または制限される場合があります。

* ユーザーは、いつでもフルアクセスを無効にし、本アプリの利用を停止できます。

当社は、ユーザーの文章が個人的かつ重要な情報であることを理解し、本アプリを安心して利用できるよう、継続的にプライバシー保護とセキュリティの改善に取り組みます。

# App Store用に特に大事な実装メモ

このポリシーに合わせるなら、実装・App Store回答もこうするのが安全です。

**絶対に「全入力を収集」しない。**
収集するのは、ユーザーがAIボタンを押して送信した文章だけ。

**App Storeの説明文にも書く。**
「AI変換のため、ユーザーが送信した文章のみを処理します。すべての入力内容を記録することはありません。」

**フルアクセス説明文を書く。**
「AI変換のためにインターネット通信を使用します。フルアクセスを許可しても、すべての入力を収集することはありません。」

**プライバシーラベルは正直に出す。**
User Content, Usage Data, Diagnostics, Identifiers あたりは該当する可能性が高いです。第三者SDKを使う場合は、そのSDKの収集内容もApp Store Connectに反映する必要があります。 Appleはアプリ本体だけでなく、統合している第三者パートナーのデータ取扱いも申告対象にしています。([Apple Developer][1])

[1]: https://developer.apple.com/app-store/app-privacy-details/?utm_source=chatgpt.com "App Privacy Details - App Store"
